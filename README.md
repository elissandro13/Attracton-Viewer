# AttractonViewer

A wikia page for visiting spots in Brazil

## Getting Started

git clone https://github.com/elissandro13/Attracton-Viewer.git

### Prerequisites

* Node
* NPM
* MongoDb


### Installing

```
npm install

```
### Running the Project

```

node app.js

```

## What I Learned

* NoSQL DN (MongoDb)
* RESTful routes
